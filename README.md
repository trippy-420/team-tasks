## üöÄ Getting Started with team-tasks

When you first open team-tasks, you will see the main dashboard. Here‚Äôs a simple guide to start your first workflow:

1. **Create a new project:** Click "New Project" and enter a name.
2. **Choose your workflow mode:** Pick Linear, DAG, or Debate depending on how your AI team should work.
3. **Add agents:** Each agent represents an AI helper. Add as many as you need by clicking "Add Agent."
4. **Define tasks:** For each agent, create tasks. These are the jobs the AI will perform.
5. **Set task order:** Arrange tasks in the workflow panel depending on your mode. In Linear mode, tasks run one after another. In DAG mode, connect tasks with arrows. In Debate mode, set agents to discuss assigned topics.
6. **Assign skills:** Choose skills or abilities for each agent to use during tasks. Skills might include language processing, decision analysis, or data fetching.
7. **Start the workflow:** Click "Run." Watch as the agents work through their tasks step by step.
8. **Monitor progress:** See detailed logs and status updates in real time.
9. **Review results:** When done, export the outcome or save the project to continue later.
